# 在你自己的 Windows 电脑上跑起来（新手零基础版）

这是你用 AI 写出的第一个应用。下面每一步都很详细，照着做就行。**不需要你会编程**，只需要会复制命令、会点"下一步"。

---

## 总览：你要做 4 件事

1. **拿到代码**（下载压缩包，解压）
2. **装 4 个免费软件**（Node、pnpm、Rust、VS 编译工具）
3. **运行一行命令**让窗口弹出来
4. （可选）打包成安装程序发给朋友

预计耗时：首次安装约 20~40 分钟（主要在下载和 Rust 首次编译），之后每次运行只要几秒。

---

## 第 1 步：拿到代码

我已经把项目打包成了 `desktop-todo.zip`。

- 在你的工作区/文件列表里找到 `desktop-todo.zip`，**下载**到你的电脑。
- 解压到一个**不含中文、不含空格**的路径，推荐：
  - `D:\desktop-todo` （在 D 盘新建文件夹 desktop-todo，把压缩包内容解压进去）
- 解压后确认里面有这些文件/文件夹：`src`、`src-tauri`、`package.json`、`README.md`、`GET_STARTED_WINDOWS.md`。

> ⚠️ 路径不要有中文和空格，否则后面命令会报错。例如 `D:\我的待办` 不行，`D:\desktop-todo` 可以。

---

## 第 2 步：安装 4 个免费软件（按顺序来）

### 2.1 Node.js（自带 npm）

1. 打开浏览器访问 https://nodejs.org
2. 点首页大大的 **LTS** 版本下载（Windows 版，文件名类似 `node-vXX.X.X-x64.msi`）
3. 双击安装，**一路点 "Next / 下一步"**，全部用默认选项即可
4. **验证是否装好**：
   - 按 `Win` 键，输入 `PowerShell`，打开"Windows PowerShell"
   - 在蓝/黑窗口里输入：
     ```
     node -v
     ```
   - 看到类似 `v20.11.0` 的版本号 → 成功 ✅
   - 如果提示"找不到命令"，**关掉 PowerShell 重新打开**再试

### 2.2 pnpm（包管理器）

仍在 PowerShell 里输入：
```
npm install -g pnpm
```
装完后验证：
```
pnpm -v
```
看到版本号（如 `9.x.x`）→ 成功 ✅

> 如果报错"无法加载 pnpm，因为在此系统上禁止运行脚本"，先执行这一句再重试：
> ```
> Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
> ```

### 2.3 Rust（编译工具，必装）

1. 浏览器访问 https://rustup.rs
2. 页面里有 `rustup-init.exe` 的下载链接，下载并双击
3. 它会打开一个黑色命令行窗口，问你安装选项，**直接按回车**（选默认 1）
4. 等待下载完成（可能几分钟），最后看到 "Rust is installed" 类似字样
5. **验证**：重新打开一个 PowerShell，输入：
   ```
   cargo --version
   ```
   看到 `cargo 1.xx.x` → 成功 ✅

### 2.4 Visual Studio 生成工具（C++ 编译环境）

1. 浏览器访问 https://visualstudio.microsoft.com/zh-hans/downloads/
2. 找到 **"Visual Studio 2022 生成工具"**（Build Tools），下载并运行
3. 安装器打开后，在"工作负载"里**勾选「使用 C++ 的桌面开发」**
4. 右下角点"安装"，等十几分钟
5. 装完即可，不需要打开 VS 写代码

### 2.5 WebView2（一般已有，检查一下）

- Windows 11 自带；较新的 Windows 10 也自带。
- 如果你不确定，访问微软官网搜"WebView2 Runtime 下载"，装一下"常青版引导程序"即可。

---

## 第 3 步：让应用跑起来 🎉

1. 打开 **PowerShell**
2. 进入你的项目目录（把路径换成你解压的位置）：
   ```
   cd D:\desktop-todo
   ```
3. 安装前端依赖（第一次较慢，要下载一会儿）：
   ```
   pnpm install
   ```
4. **启动应用**（关键一步）：
   ```
   pnpm tauri dev
   ```
5. 第一次会**编译 Rust**（几分钟，黑窗口刷很多字是正常的），编译完后会自动弹出一个**透明的小窗口**贴在桌面上 → 成功！🎉

> 窗口是透明的、置顶的。鼠标按住顶部栏可以拖动窗口，右下角斜纹可以调整大小；点击列表空白处直接输入待办；点圆圈打勾；点 🔧 编辑详情；拖左侧 `⋮⋮` 排序；右上角 ⚙ 是设置。

---

## 第 4 步：打包成安装程序（发给朋友）

应用做好后，想发给朋友：

```
pnpm tauri build
```

完成后，安装包在这里：
```
D:\desktop-todo\src-tauri\target\release\bundle\
```
里面有 `.msi` 或 `.exe` 安装程序，发给朋友双击安装即可。

> 朋友首次安装会被 Windows 拦截（SmartScreen 提示"已保护你的电脑"），点"详细信息 → 仍要运行"就行。这是正常的，不影响使用，不用花钱买证书。

## 常见问题

**Q：命令提示"不是内部或外部命令"？**
→ 软件没装好，或装完后没**重开 PowerShell**。重开再试。

**Q：`pnpm tauri dev` 卡住/报错？**
→ 第一次编译 Rust 要几分钟，耐心等。若报"LINK"相关错误，通常是 2.4 的 VS 生成工具没装或没勾「C++ 桌面开发」，回去补装。

**Q：窗口是白的/不透明？**
→ 正常应是透明。若你系统主题导致看不清，打开 ⚙ 设置调低"窗口透明度"。

**Q：改了代码怎么看效果？**
→ `pnpm tauri dev` 已经在跑的话，改 `src/` 下的文件保存后会自动刷新。改 `src-tauri/` 下的 Rust 代码需要停掉（Ctrl+C）再重新 `pnpm tauri dev`。

**Q：我不想装这么多东西，只想先看界面？**
→ 可以！只装 2.1 和 2.2（Node + pnpm），然后在项目目录运行：
> ```
> pnpm dev
> ```
> 浏览器打开 http://localhost:1420 就能看界面（用浏览器 localStorage 存数据，没有透明窗口效果）。
